# Date-a-Scientist

An academic data analysis and machine learning project exploring OKCupid user profile data, provided by Codecademy.

The project analyzes user demographics (age, income, height, religion, education) through visualizations, then applies binary classification models — Logistic Regression, K-Nearest Neighbors, and Random Forest — to predict a user's sex based on their profile features.

Built with Python using Pandas, NumPy, Seaborn, Matplotlib, and Scikit-learn in a Jupyter Notebook environment.
